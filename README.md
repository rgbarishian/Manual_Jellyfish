# Manual_Jellyfish
Computational Methods to identify Jellyfish Galaxies.

FITS images are downloaded from HST datasets
SExtractor is run to get FLUX_BEST, KRON_RADIUS, X_WORLD, Y_WORLD, A_WORLD, B_WORLD

Input:
SExtractor.dat file into Python script

Output:
.reg, DS9 regions files. that overlay images
